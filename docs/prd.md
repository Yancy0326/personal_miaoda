# 清爽科技感个人展示网页需求文档

## 1. 网站概述

### 1.1 网站名称\n个人展示网页

### 1.2 网站描述

一个专为AI创作者设计的清爽科技感个人展示网页，用于展示专业能力和作品，建立个人品牌影响力。

## 2. 设计风格

### 2.1 视觉风格

- 配色方案：深色背景配合高对比度亮色元素，营造科技感与专业感
- 字体选择：大号无衬线字体突出名称，现代简约字体用于正文
- 视觉细节：适度圆角设计，平滑过渡动画效果，悬停微交互
- 整体布局：一屏式布局，卡片式分组，信息层级清晰

## 3. 功能模块

### 3.1 基础信息区（顶部居中）

- 个人名称（大号无衬线字体）
- 圆形/方形头像/Logo（80-100px直径）
- 个人标签云（2-3个标签，如"AI产品设计师｜交互专家｜创意技术爱好者"）

### 3.2 职业定位区（头像下方）

- 一句话个人介绍（加粗突出）
- 当前职业方向
- 核心领域标签（3-4个，如"生成式AI｜用户体验｜交互设计"）

### 3.3 能力与特长区（左侧或右侧）

- 技能展示：5-7项核心技能（进度条或星级形式）
- 价值主张：3-4点"我能提供的价值"

### 3.4 代表作品区（中部偏下）

- 横向卡片式展示2-3个典型项目
- 每个项目包含：缩略图、项目名称、一句话说明、技术标签

### 3.5 当前项目区（作品区下方）

- "我正在做的事"：当前专注的项目或研究描述
- "我关注的方向"：2-3个前沿领域关键词

### 3.6 合作方式区（右侧或左侧）

- 可合作内容：3-4个合作方向（如"AI产品咨询｜交互设计指导｜创意技术开发"）
- 可接受形式：远程协作/线下交流/项目合作等选项

### 3.7 联系方式区（底部左侧）

- 微信二维码+微信号
- 邮箱地址（带mailto链接）
- Bilibili主页链接和二维码
- 小红书主页链接和二维码
- 社交媒体图标链接（2-3个平台，如GitHub、LinkedIn、知乎、Bilibili、小红书）

### 3.8 个性亮点区（底部右侧）

- 个人价值观或特色宣言
- 独特视觉元素（个人签名、代表性图标）

## 4. 技术要求

### 4.1 响应式设计

- 使用Tailwind CSS或纯CSS实现移动端自适应
- 所有内容区域支持点击编辑
- 一屏式布局，所有内容单屏内完整展示

### 4.2 交互特性

- 适度微交互动画（悬停效果、平滑过渡）
- 留白合理，信息层级清晰

### 4.3 开发规范

- 提供完整的HTML/CSS/JS代码
- 代码结构清晰，添加必要注释
- 确保页面加载速度快，无冗余代码
