# 个人展示网页使用说明

## 概述

这是一个清爽科技感的个人展示网页，专为AI创作者设计，用于展示专业能力和作品，建立个人品牌影响力。

**✨ 特色功能：所有内容都可以在线编辑，无需修改代码！**

## 功能特点

### 1. 设计风格
- ✨ 深色科技感主题，蓝紫渐变配色
- 🎨 现代简约设计，卡片式布局
- 💫 流畅的动画效果和微交互
- 📱 完全响应式设计，支持移动端和桌面端
- 🖊️ **所有内容可在线编辑，实时保存**

### 2. 功能模块

#### 基础信息区
- 个人头像展示
- 姓名和标签云
- 支持自定义个人标签

#### 职业定位区
- 个人介绍
- 当前职业方向
- 核心领域标签展示

#### 能力与特长区
- 技能进度条展示（5项核心技能）
- 价值主张列表
- 可视化技能水平

#### 代表作品区
- 项目卡片展示（支持2-3个项目）
- 项目图片、名称、描述
- 技术标签展示
- 悬停效果

#### 当前项目区
- 正在做的事情描述
- 关注的方向标签

#### 合作方式区
- 可合作内容展示
- 可接受的合作形式

#### 联系方式区
- 邮箱联系（支持mailto链接）
- 微信二维码展示
- 社交媒体链接（Bilibili、小红书、GitHub、LinkedIn）

#### 个性亮点区
- 个人价值观展示
- 个人信条
- 特色签名

## 如何使用在线编辑功能

### 🎯 快速开始

1. **打开网页**：在浏览器中打开您的个人展示网页
2. **悬停编辑**：将鼠标悬停在任何内容区域，会出现"编辑"按钮
3. **点击编辑**：点击编辑按钮，在弹出的对话框中修改内容
4. **保存修改**：点击"保存"按钮，您的修改会立即生效
5. **自动保存**：所有修改会自动保存到浏览器本地存储

### 📝 可编辑的内容区域

#### 1. 基础信息区
- **位置**：页面顶部
- **编辑方式**：悬停后点击右上角"编辑基础信息"按钮
- **可编辑内容**：
  - 姓名
  - 职位
  - 头像URL
  - 个人标签（用逗号分隔）
  - 个人介绍
  - 当前职位
  - 核心领域（用逗号分隔）

#### 2. 核心技能
- **位置**：左侧栏第一个卡片
- **编辑方式**：
  - 悬停在技能条上，点击"编辑"图标修改
  - 点击"+"按钮添加新技能
  - 点击"垃圾桶"图标删除技能
- **可编辑内容**：
  - 技能名称
  - 技能等级（0-100）

#### 3. 价值主张
- **位置**：左侧栏第二个卡片
- **编辑方式**：
  - 悬停在价值条目上，点击"编辑"图标修改
  - 点击"+"按钮添加新价值
  - 点击"垃圾桶"图标删除价值
- **可编辑内容**：价值描述文本

#### 4. 合作方式
- **位置**：左侧栏第三个卡片
- **编辑方式**：悬停后点击右上角"编辑"按钮
- **可编辑内容**：
  - 可合作内容（用逗号分隔）
  - 可接受形式（用逗号分隔）

#### 5. 代表作品
- **位置**：右侧栏第一个卡片
- **编辑方式**：
  - 悬停在项目卡片上，点击"编辑"图标修改
  - 点击"添加项目"按钮添加新项目
  - 点击"垃圾桶"图标删除项目
- **可编辑内容**：
  - 项目名称
  - 项目描述
  - 技术标签（用逗号分隔）
  - 项目图片URL

#### 6. 当前专注
- **位置**：右侧栏第二个卡片
- **编辑方式**：悬停后点击右上角"编辑"按钮
- **可编辑内容**：
  - 我正在做的事
  - 我关注的方向（用逗号分隔）

#### 7. 联系方式
- **位置**：底部左侧卡片
- **编辑方式**：悬停后点击右上角"编辑"按钮
- **可编辑内容**：
  - 邮箱
  - 微信号
  - 微信二维码URL
  - Bilibili主页
  - 小红书主页
  - GitHub主页
  - LinkedIn主页

#### 8. 个人信条
- **位置**：底部右侧卡片
- **编辑方式**：悬停后点击右上角"编辑"按钮
- **可编辑内容**：
  - 个人价值观
  - 个人签名

### 💾 数据保存说明

- **自动保存**：每次编辑后，数据会自动保存到浏览器的localStorage
- **手动保存**：点击右上角的"保存所有修改"按钮可以手动保存
- **数据持久化**：只要不清除浏览器缓存，您的数据会一直保留
- **跨设备同步**：数据保存在本地，不同设备需要分别编辑

### ⚠️ 注意事项

1. **清除缓存**：清除浏览器缓存会导致所有编辑内容丢失
2. **备份数据**：建议定期导出浏览器的localStorage数据作为备份
3. **图片URL**：确保图片URL可以公开访问，否则图片无法显示
4. **逗号分隔**：多个标签、领域等内容需要用英文逗号分隔

## 自定义指南

### 方式一：在线编辑（推荐）

**无需修改代码，直接在网页上编辑所有内容！**

详见上方"如何使用在线编辑功能"章节。

### 方式二：修改默认数据（高级）

如果您想修改初始默认数据，可以编辑 `src/pages/Home.tsx` 文件中的 `defaultData` 对象：

```typescript
// 1. 修改基础信息
const [personalInfo] = useState({
  name: "你的名字",
  title: "你的职位",
  avatar: "你的头像URL",
  tags: ["标签1", "标签2", "标签3"],
  bio: "你的一句话介绍",
  currentRole: "当前职位 @ 公司名称",
  fields: ["领域1", "领域2", "领域3", "领域4"]
});

// 2. 修改技能
const [skills] = useState([
  { name: "技能名称", level: 95 },
  // 添加更多技能...
]);

// 3. 修改价值主张
const [values] = useState([
  "价值点1",
  "价值点2",
  // 添加更多价值点...
]);

// 4. 修改项目
const [projects] = useState([
  {
    name: "项目名称",
    description: "项目描述",
    tags: ["标签1", "标签2"],
    image: "项目图片URL"
  },
  // 添加更多项目...
]);

// 5. 修改联系方式
const [contacts] = useState({
  email: "your@email.com",
  wechat: "你的微信号",
  wechatQR: "微信二维码URL",
  bilibili: "B站主页链接",
  xiaohongshu: "小红书主页链接",
  github: "GitHub链接",
  linkedin: "LinkedIn链接"
});

// 6. 修改个人信条
const [motto] = useState({
  text: "你的价值观",
  signature: "你的签名"
});
```

## 性能优化建议

1. **图片优化**: 使用WebP格式的图片，建议尺寸：
   - 头像: 200x200px
   - 项目图片: 800x500px
   - 二维码: 200x200px

2. **加载速度**: 
   - 使用CDN托管图片
   - 启用图片懒加载
   - 压缩图片文件大小

3. **SEO优化**:
   - 已配置页面标题和描述
   - 使用语义化HTML标签
   - 添加适当的meta标签

## 常见问题

### Q: 如何更换头像？
A: 修改 `personalInfo.avatar` 的值为你的头像图片URL。

### Q: 如何添加更多技能？
A: 在 `skills` 数组中添加新的对象，包含 `name` 和 `level` 属性。

### Q: 如何修改页面标题？
A: 编辑 `index.html` 文件中的 `<title>` 标签。

### Q: 二维码如何生成？
A: 可以使用在线二维码生成工具，或者使用API服务如 `https://api.qrserver.net/`。

### Q: 如何禁用深色模式？
A: 在 `src/App.tsx` 中移除 `document.documentElement.classList.add('dark');` 这行代码。
