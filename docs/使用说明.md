# 个人展示网页使用说明

## 概述

这是一个清爽科技感的个人展示网页，专为AI创作者设计，用于展示专业能力和作品，建立个人品牌影响力。

## 功能特点

### 1. 设计风格
- ✨ 深色科技感主题，蓝紫渐变配色
- 🎨 现代简约设计，卡片式布局
- 💫 流畅的动画效果和微交互
- 📱 完全响应式设计，支持移动端和桌面端

### 2. 功能模块

#### 基础信息区
- 个人头像展示
- 姓名和标签云
- 支持自定义个人标签

#### 职业定位区
- 个人介绍
- 当前职业方向
- 核心领域标签展示

#### 能力与特长区
- 技能进度条展示（5项核心技能）
- 价值主张列表
- 可视化技能水平

#### 代表作品区
- 项目卡片展示（支持2-3个项目）
- 项目图片、名称、描述
- 技术标签展示
- 悬停效果

#### 当前项目区
- 正在做的事情描述
- 关注的方向标签

#### 合作方式区
- 可合作内容展示
- 可接受的合作形式

#### 联系方式区
- 邮箱联系（支持mailto链接）
- 微信二维码展示
- 社交媒体链接（Bilibili、小红书、GitHub、LinkedIn）

#### 个性亮点区
- 个人价值观展示
- 个人信条
- 特色签名

## 自定义指南

### 修改个人信息

打开 `src/pages/Home.tsx` 文件，找到以下部分进行修改：

```typescript
// 1. 修改基础信息
const [personalInfo] = useState({
  name: "你的名字",
  title: "你的职位",
  avatar: "你的头像URL",
  tags: ["标签1", "标签2", "标签3"],
  bio: "你的一句话介绍",
  currentRole: "当前职位 @ 公司名称",
  fields: ["领域1", "领域2", "领域3", "领域4"]
});

// 2. 修改技能
const [skills] = useState([
  { name: "技能名称", level: 95 },
  // 添加更多技能...
]);

// 3. 修改价值主张
const [values] = useState([
  "价值点1",
  "价值点2",
  // 添加更多价值点...
]);

// 4. 修改项目
const [projects] = useState([
  {
    name: "项目名称",
    description: "项目描述",
    tags: ["标签1", "标签2"],
    image: "项目图片URL"
  },
  // 添加更多项目...
]);

// 5. 修改联系方式
const [contacts] = useState({
  email: "your@email.com",
  wechat: "你的微信号",
  wechatQR: "微信二维码URL",
  bilibili: "B站主页链接",
  xiaohongshu: "小红书主页链接",
  github: "GitHub链接",
  linkedin: "LinkedIn链接"
});

// 6. 修改个人信条
const [motto] = useState({
  text: "你的价值观",
  signature: "你的签名"
});
```

### 修改配色方案

如需修改配色，编辑 `src/index.css` 文件中的颜色变量：

```css
.dark {
  --primary: 217 91% 60%;        /* 主色调 */
  --secondary: 280 100% 70%;     /* 次要色 */
  --background: 220 20% 8%;      /* 背景色 */
  --card: 220 20% 12%;           /* 卡片背景 */
  /* 更多颜色变量... */
}
```

### 添加更多项目

在 `projects` 数组中添加新的项目对象：

```typescript
{
  name: "新项目名称",
  description: "项目描述，一句话说明项目亮点",
  tags: ["技术1", "技术2", "技术3"],
  image: "项目展示图片URL"
}
```

### 修改社交媒体链接

在 `contacts` 对象中更新相应的链接：

```typescript
const [contacts] = useState({
  email: "your@email.com",
  wechat: "your_wechat_id",
  wechatQR: "https://your-qr-code-url.com/wechat.png",
  bilibili: "https://space.bilibili.com/your_id",
  xiaohongshu: "https://www.xiaohongshu.com/user/profile/your_id",
  github: "https://github.com/your_username",
  linkedin: "https://linkedin.com/in/your_profile"
});
```

## 性能优化建议

1. **图片优化**: 使用WebP格式的图片，建议尺寸：
   - 头像: 200x200px
   - 项目图片: 800x500px
   - 二维码: 200x200px

2. **加载速度**: 
   - 使用CDN托管图片
   - 启用图片懒加载
   - 压缩图片文件大小

3. **SEO优化**:
   - 已配置页面标题和描述
   - 使用语义化HTML标签
   - 添加适当的meta标签

## 常见问题

### Q: 如何更换头像？
A: 修改 `personalInfo.avatar` 的值为你的头像图片URL。

### Q: 如何添加更多技能？
A: 在 `skills` 数组中添加新的对象，包含 `name` 和 `level` 属性。

### Q: 如何修改页面标题？
A: 编辑 `index.html` 文件中的 `<title>` 标签。

### Q: 二维码如何生成？
A: 可以使用在线二维码生成工具，或者使用API服务如 `https://api.qrserver.net/`。

### Q: 如何禁用深色模式？
A: 在 `src/App.tsx` 中移除 `document.documentElement.classList.add('dark');` 这行代码。
